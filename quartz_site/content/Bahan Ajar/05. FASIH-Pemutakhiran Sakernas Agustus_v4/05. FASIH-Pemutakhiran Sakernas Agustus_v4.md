# MEMBUKA APLIKASI DAN UNDUH TEMPLATE & VALIDASI

### PERIODE

#### Status Assignment:

- *a. OPEN** : status awal sampel yang dialokasikan kepada PPL dan belum dilakukan submit
- *b. SUBMIT** : status sampel yang sudah dikerjakan oleh PPL yang kemudian disubmit dan mendapat respon sukses dari server
- *c. REJECT** : merupakan status sampel yang sudah pernah di-submit ke server tetapi setelah diperiksa ternyata ditolak oleh PML sehingga dikembalikan ke PPL
- *d. PENDING** : status sampel yang di-submit oleh PPL tetapi belum mendapat respon dari server karena masih menunggu koneksi internet atau antrian masuk ke server
- *e. APPROVED** : status sampel yang pernah di-submit oleh PPL ke server dan setelah diperiksa, sudah dinyatakan sesuai oleh PML/admin kab/kota

**Periode,** menampilkan periode survei,

# DAFTAR WILAYAH (SLS)

Setiap ada **perubahan assignments** (**tambahan assignments / approval assignments**) akan muncul notifikasi untuk **unduh perubahan assignments**

#### Keterangan:

- *Buka** : menampilkan E-form SAKERNAS AGS 2026- PEMUTAKHIRAN
- *Sinkronisasi Assignment Ini**: khusus untuk assignment **selain** status OPEN/DRAFT fungsinya untuk mendapatkan STATUS dan mengunduh data dari server. Khusus untuk assignment OPEN/DRAFT, fungsinya untuk mendapatkan status assignment tanpa mengunduh data.
- *Assign** : melakukan alokasi sampel ke PPL oleh PML
- *Restore**: memulihkan kembali isian data wawancara sesuai kondisi pada versi tanggal wawancara yang dipilih
- *Info Assignment:** menampilkan version code lokal dan server serta ukuran data
- *Hapus** : menghapus assignment sampel yang telah ditambahkan secara manual oleh PPL dan berstatus belum disubmit

### FITUR DI E-FORM

**Menu (burger icon),**

menampilkan versi form gear, template dan rule validasi. Selain itu menampilkan section per blok pertanyaan.

**Section**, untuk menampilkan pertanyaan di blok tersebut

## FITUR KUESIONER

- *Lokasi,** menampilkan dimana lokasi perangkat saat ini
- *Bantuan**, akan diarahkan pada halaman pusat bantuan Fasih
- *Muat Ulang**, digunakan untuk *load* ulang data, hanya digunakan apabila disarankan oleh tim fasih jika ada kendala
- *Simpan Sementara,** untuk menyimpan sementara hasil pendataan sebelum beralih ke halaman berikutnya. Tersedia opsi untuk menyimpan data di local atau dikirim ke server

**Lembar Kerja**, tools untuk membantu mencatat informasi tambahan selama wawancara

**Kalkulator**, tools untuk membantu perhitungan

# TARIK SAMPEL di FASIH-SM (4)

#### TARIK SAMPEL OFFLINE di FASIH MOBILE – PPL

#### (Khusus wilayah blankspot yang terdaftar di awal)

Hal – hal yang harus dilakukan sebelum memasuki wilayah blankspot agar dapat dilakukan Tarik sampel offline (dilakukan saat masih terhubung sinyal internet yang baik) :

1. SLS **sudah terdaftar offline sampling** dan disetujui MSSD sebagai wilayah blankspot di awal pendataan
2. Mendapatkan **tag "OFFLINE SAMPLING"** di halaman "Daftar Wilayah"
3. Pada surveI SAKERNAS PEMUTAKHIRAN di Aplikasi FASIH Mobile, PPL **sudah mengunduh seluruh daftar assignment pemutakhiran** untuk wilayah SLS blankspot tersebut
4. Pada survey SAKERNAS PENDATAAN di Aplikasi FASIH Mobile, PPL sudah selesai melakukan seluruh sync sampai PPL **sudah membuka halaman "Daftar Assignment" survey SAKERNAS PENDATAAN**, walaupun halaman "Daftar Assignment" belum terisi assignment apapun.

#### TARIK SAMPEL OFFLINE di FASIH MOBILE – PPL (2)

#### 2. Mendapatkan tag "OFFLINE SAMPLING" di halaman "Daftar Wilayah"

#### Cara mendapatkan tag merah offline sampling :

- *a. Pastikan terhubung sinyal internet**
- *b. Buka halaman daftar wilayah**
- *c. Pada halaman daftar wilayah, klik titik tiga di kanan atas, klik "check offline sampling"**
- *d. Back, ke halaman daftar periode**
- *e. Masuk lagi ke halaman daftar wilayah, klik titik tiga di kanan atas, klik "check offline sampling"**
- *f. Back, ke halaman daftar periode**
- *g. Masuk lagi ke halaman daftar wilayah, klik titik tiga di kanan atas, pastikan sudah muncul tag merah OFFLINE SAMPLING di setiap blok sensus yang sudah terdaftar offline sampling**

#### TARIK SAMPEL OFFLINE di FASIH MOBILE – PPL (3)

**3. Pada survey SAKERNAS PEMUTAKHIRAN di Aplikasi FASIH Mobile, PPL sudah mengunduh seluruh daftar assignment pemutakhiran untuk wilayah SLS blankspot tersebut**

#### TARIK SAMPEL OFFLINE di FASIH MOBIL – PPL (4)

4. Pada survey SAKERNAS PENDATAAN di Aplikasi FASIH Mobile, PPL sudah selesai melakukan seluruh sync sampai PPL sudah membuka halaman "Daftar Assignment" survey SAKERNAS PENDATAAN, walaupun halaman "Daftar Assignment" belum terisi assignment apapun.

#### TARIK SAMPEL OFFLINE di FASIH MOBILE – PPL (5)

#### Poin-poin penting terkait SLS dengan Tarik sampel offline :

- *Walaupun dilakukan dalam kondisi offline dan belum disubmit, seluruh assignment pemutakhiran tetap harus clean validasi.**
- *PML telah melakukan pemeriksaan hasil pemutakhiran SLS tersebut bersama PPL menggunakan HP PPL**
- *Pastikan setelah selesai pemutakhiran, melakukan backup data terlebih dahulu sebelum melakukan tarik sampel offline**
- *Setelah melakukan backup data, silahkan melakukan Tarik sampel offline, saat proses Tarik sampel offline mohon ditunggu jika layar menjadi putih sampai muncul popup sukses.**
- *Pastikan setelah kembali ke wilayah yang terdapat sinyal, submit semua assignment pemutakhiran dan pendataan dari SLS Offline sampling tersebut**
